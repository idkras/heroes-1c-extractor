# PRD: Извлечение актов, накладных и счетов-фактур из 1CD файла

## Цель проекта
Создать инструмент для извлечения реальных документов (акты, накладные, счета-фактуры) из файла 1CD для анализа цветочков, а НЕ справочников.

## Критические требования

### Основные документы для извлечения:
1. **Акты** - документы о выполненных работах/услугах
2. **Накладные** - документы о движении товаров
3. **Счета-фактуры** - документы для налогового учета

### НЕ извлекать:
- Справочники (контрагенты, товары, номенклатура)
- Служебные таблицы
- Конфигурационные данные

## Технические требования

### Входные данные:
- Файл 1CD (база данных 1С)
- Автоматическое определение структуры базы

### Выходные данные:
- Структурированные данные актов, накладных и счетов-фактур
- Формат: JSON/CSV для дальнейшего анализа
- Только реальные документы, не справочники

### Функциональные требования:
1. Автоматическое сканирование структуры 1CD файла
2. Идентификация таблиц с документами (не справочниками)
3. Извлечение данных из найденных документов
4. Фильтрация по типам документов (акты, накладные, счета-фактуры)
5. Экспорт в читаемый формат

### Технический стек:
- Python для обработки 1CD файлов
- Библиотеки для работы с 1С форматами
- Автоматическое определение структуры
- Без GUI интерфейса

## Критерии успеха:
- Извлечены только реальные документы
- Нет справочников в выходных данных
- Структурированный формат для анализа
- Автоматическая работа без ручного вмешательства
